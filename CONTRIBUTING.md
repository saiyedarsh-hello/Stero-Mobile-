# Contributing to Stero

First off, thank you for considering contributing to Stero! It's people like you that make Stero such a great tool.

## How Can I Contribute?

### Reporting Bugs

This section guides you through submitting a bug report for Stero. Following these guidelines helps maintainers and the community understand your report, reproduce the behavior, and find related reports.

* Use the provided Bug Report issue template.
* Check if the bug has already been reported.
* Provide clear steps to reproduce the issue.

### Suggesting Enhancements

This section guides you through submitting an enhancement suggestion for Stero, including completely new features and minor improvements to existing functionality.

* Use the provided Feature Request issue template.
* Provide a compelling reason why this enhancement would be useful to most Stero users.

### Pull Requests

* Fill in the required template.
* Do not include issue numbers in the PR title.
* Follow the styleguides of the project.
* After you submit your pull request, verify that all status checks are passing.

## Development Setup

1. Fork and clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.

Thank you!
